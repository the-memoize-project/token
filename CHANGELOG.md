# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-XX

### Added

- Initial release of @the-memoize-project/token
- 108 design tokens across 10 categories:
  - Color tokens (42): Master, Primary, Semantic (Complete, Success, Warning, Danger, Info), Menu, Pure
  - Spacing tokens (21): Margin/gap scale and inset padding scale
  - Font family tokens (2): Base and highlight families
  - Font size tokens (11): From xxxs (12px) to giant (96px)
  - Font weight tokens (3): Regular, medium, bold
  - Line height tokens (7): From default to xxl
  - Border radius tokens (7): From none to circular
  - Border width tokens (5): From none to heavy
  - Shadow tokens (4): Four elevation levels
  - Opacity tokens (5): Five transparency levels
- Full light/dark mode support using CSS `light-dark()` function
- TypeScript declarations for CSS imports
- Comprehensive documentation (README, ARCHITECTURE, CONTRIBUTING)
- MIT License
- Biome configuration for code quality
- Conventional Commits setup with commitlint
- Git hooks with Husky and lint-staged
